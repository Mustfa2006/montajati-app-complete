## 🚀 **خطوات النشر (5 دقائق):**

### **الطريقة السريعة:**
```bash
# 1. بناء الموقع
cd frontend
flutter build web --release

# 2. النشر على Netlify
# - اذهب إلى netlify.com
# - اسحب مجلد build/web
# - احصل على الرابط
```

### **الطريقة المتقدمة (GitHub + Netlify):**
```bash
# 1. رفع الكود إلى GitHub
git add .
git commit -m "إضافة دعم الويب"
git push

# 2. ربط Netlify مع GitHub
# - نشر تلقائي عند كل تحديث
```
